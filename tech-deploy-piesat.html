一、4 套 Schema JSON-LD 代码

Schema 是给搜索引擎和 AI 引擎看的"身份信息"——告诉它们这个网站是谁、有什么产品、文章讲什么、常见问题是什么。以下代码全部粘贴到对应页面的 <head> 标签内。

Schema 1 · Organization（粘贴到网站首页 head 中）

三、robots.txt 检查与修正

目前 5 个子域名（factory/oa/edu/znpt/lvyou）百度收录为 0，需要排查是否被 robots.txt 误屏蔽。

第一步：排查现有限制
  1. 访问各子域名的 /robots.txt（如 https://engine.piesat.cn/robots.txt）
  2. 检查是否有 Disallow: / 拦截了所有爬虫
  3. 检查是否有 User-agent: Baiduspider + Disallow: / 专门拦截百度
第二步：按以下模板统一配置

对外公开的子域名（www / engine / engine-dev / factory / edu / znpt / lvyou），根目录 robots.txt：

User-agent: * Allow: / Disallow: /admin/ Disallow: /api/ Sitemap: https://www.piesat.cn/sitemap.xml

内部系统（oa），根目录 robots.txt：

User-agent: * Disallow: /
四、sitemap.xml 生成与提交
第 1 步：生成 sitemap.xml
  1. 方式 A：如果 CMS 自带 sitemap 功能（WordPress + Yoast / Rank Math）→ 后台点击"生成 sitemap"
  2. 方式 B：使用在线工具（如 xml-sitemaps.com）→ 输入 piesat.cn → 下载生成的 sitemap.xml
  3. 上传 sitemap.xml 到网站根目录（如 /var/www/html/sitemap.xml）
第 2 步：提交到百度站长平台
  1. 登录百度站长平台（ziyuan.baidu.com）→ 选择已验证的 piesat.cn
  2. 左侧"数据引入"→"sitemap"→ 填入 https://www.piesat.cn/sitemap.xml → 提交
第 3 步：每次发布新文章后手动推送（对接人操作）
  1. 百度站长平台 →"普通收录"→"手动提交"
  2. 粘贴新文章 URL → 提交（每次约 20 秒）
